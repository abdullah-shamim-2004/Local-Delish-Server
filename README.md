Welcome to the **LocalDelish** backend ‚Äî a modern RESTful API built for managing restaurant reviews, favorites, and user data for the **LocalDelish** food discovery platform.

---

##  Project Overview

The backend is designed to:
- Handle **user authentication** securely (with Firebase).
- Manage **restaurant reviews**, including CRUD operations.
- Allow users to **add and manage their favorite dishes**.
- Serve data efficiently to the **LocalDelish Client**.

This backend is built using **Node.js, Express, and MongoDB** .
